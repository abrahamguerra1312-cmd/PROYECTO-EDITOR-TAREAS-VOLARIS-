# ✈️ Volaris PDF Processor

Aplicación para limpiar y preparar documentación técnica PDF para impresión, optimizada para aerolíneas Volaris.

## 🎯 Funcionalidades

- **Identificación automática de tareas**: Detecta Task Cards, Engineering Orders, Daily Checks y Weekly Checks
- **Eliminación de contenido innecesario**: Remueve hojas de control, metadatos, tablas vacías
- **Optimización para impresión doble cara**: Añade páginas en blanco automáticamente
- **Procesamiento por lotes**: Permite procesar múltiples archivos simultáneamente
- **Conservación de nombres**: Los archivos procesados mantienen su nombre original con una "F" al final
